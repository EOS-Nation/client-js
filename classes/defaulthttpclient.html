Class DefaultHttpClient

The DefaultHttpClient is responsible of performing the actual HTTP calls transforming the parameters into an actual HTTP request and turning back the HTTP response into a return value.

All methods are asynchronous and returns a response object when successufl and throws a DfuseError when an error is encountered.
